## Tech Stack

### Frontend
- ReactJS
- React Router 
- Semantic UI
- Cloudinary
- Apollo
- Material UI

### Backend
- Node.js
- GraphQL
- MongoDB
- JWT

### Functionalities Implemented
- User
    1. Login/Signup 
    2. JWT authentication
- Post
    1. Create or delete posts
    2. View posts from all users
    3. Like and unlike posts
    4. Comment on posts

## Deployment

Deployed backend using Heroku and frontend using Netlify.
